## Tujuan

AI memastikan UI terlihat profesional, modern, rapi, dan mudah digunakan.

## Prinsip UI

* Gunakan spacing konsisten.
* Gunakan hierarchy visual yang jelas.
* Button utama harus jelas.
* Hindari tampilan penuh teks tanpa struktur.
* Gunakan card, badge, progress bar, dan section heading.
* Dashboard harus mudah dinavigasi.
* Jangan membuat warna terlalu ramai.
* Gunakan desain responsive mobile dan desktop.

## Komponen Wajib LMS

1. Navbar
2. Sidebar dashboard
3. Course card
4. Progress bar
5. Materi list
6. Empty state
7. Loading skeleton/spinner
8. Error message
9. Success animation
10. Certificate card

## Animasi Penyelesaian Materi

Saat materi selesai:

* Tampilkan animasi singkat.
* Bisa berupa checkmark, confetti, progress completion, atau success modal.
* Jangan terlalu lama.
* Setelah animasi, redirect ke dashboard/homepage.
* Jika semua materi selesai, redirect ke sertifikat.

## Copywriting UI

Gunakan bahasa Indonesia yang jelas:

* "Materi berhasil diselesaikan"
* "Progress belajar kamu telah diperbarui"
* "Lanjutkan pembelajaran"
* "Selamat, kamu telah menyelesaikan semua materi"
* "Sertifikat kamu sudah tersedia"

## Responsive

Pastikan:

* Sidebar bisa collapse atau menjadi drawer di mobile.
* Card tidak pecah di layar kecil.
* Form nyaman digunakan di mobile.
* Button cukup besar untuk disentuh.
* Tabel punya horizontal scroll jika perlu.
